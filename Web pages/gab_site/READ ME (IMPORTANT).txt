ano copy nyo nlng yung "pao_profile.html" dito sa folder nyo then edit nyo nlng yung mga contents

Mga ieedit nyo:
    Main profile page
    Home screen icon header
    All profiles icon header (which is the same lang sa home screen kaso ang pinagkaiba nya ay nasa isa pang separate webpage)

tignan nyo nlng yung saakin for references sa mga codes

yung "script base.js" at "style base.cs" eto yung javascript at css ng buong website natin, gagamitin nyo rin nya sa each profiles nyo (see my profile code for the reference)

To deploy/run yung website either open with your browser or open nyo sya sa Visual code. Download nyo yung plugin na "Live Server", after nyang ma-install, right click nyo yung html then click "Open with Live Server". may mag pa-prompt lang na public access tas mag agree lang kayo. Mag oopen si Visual code ng new tab sa brower nyo para makita yung website while nag eedit kayo ng code. I-save nyo lang yung edits then refresh yung tab na'yon
